📡 Endpoints Disponibles

Base URL: http://localhost/9amm/IA/apis/

🔒 auth.php (Con autenticación de sesión)

Método Descripción Requiere Auth
POST Login: {"login": true, "usuario": "admin", "contrasena": "..."} No
POST Logout: {"logout": true} Sí
GET Listar usuarios Sí
POST Crear usuario (no incluir id) Sí
PUT Actualizar usuario (incluir id) Sí
DELETE Eliminar usuario: ?id=X Sí

usuarios.php (Sin autenticación de sesión)

Método Descripción Auth
GET Listar usuarios (requiere rol=admin en header o body) rol param
POST Crear usuario (requiere rol=admin) rol param
PUT Actualizar usuario (requiere rol=admin) rol param
DELETE Eliminar usuario (requiere rol=admin) rol param
// Ejemplo: Login con fetch fetch('auth.php', { method: 'POST', credentials: 'include', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ login: true, usuario: 'admin', contrasena: 'admin123' }) })
